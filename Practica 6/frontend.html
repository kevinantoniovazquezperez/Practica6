Front-end

Definicion y fundamentos: En diseño de software el front end es la parte del software que interactúa con los usuarios y el back end es la parte que procesa la entrada desde el front end. La separación del sistema en front ends y back ends es un tipo de abstracción que ayuda a mantener las diferentes partes del sistema separadas. Un desarrollador Front End debe tener conocimiento sobre HTML, CSS y JavaScript y ser capaz de utilizar algunos frameworks o librerías para crear todo tipo de interfaces de usuarios.
